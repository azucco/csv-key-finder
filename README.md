# node-cli-app (csv key finder)
Node.js cli app.

Example
======================
Csv file example:
```console
1,Rossi,Fabio,01/06/1990;
2,Gialli,Alessandro,02/07/1989;
3,Verdi,Alberto,03/08/1987;
```
Command example:
```console
$ search ./file.csv 1 Rossi
```
Output example:
```console
1,Rossi,Fabio,01/06/1990;
```
